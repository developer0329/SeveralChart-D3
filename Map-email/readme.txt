
For sending email

	1. Installation and Configuration
		
		1) Install PEAR Framework
			https://pear.php.net/manual/en/installation.getting.php
		2) Install HTTP2, Mail and Net_SMTP packages for PEAR
			
			Run following Statements On Command line
				pear install HTTP2-1.1.1
				pear install Mail-1.3.0
				pear install Net_SMTP
			
		More information for Install Packages and PEAR
			https://www.youtube.com/watch?v=gR3YFgg79mw
